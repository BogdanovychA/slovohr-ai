# Slovohr.AI (Словограй)

Проєкт створений в межах бета-тесту Lapathoniia — першого українського LLM інференс-провайдера.

## Налаштування змінних середовища

Проєкт використовує два типи конфігураційних файлів:

1.  **Конфігурація застосунку**: Створіть файл `src/assets/.env` на основі `src/assets/.env.example` та додайте ваш API ключ Lapathoniia.
2.  **Конфігурація Docker** (лише для Docker): Створіть файл `.env` у корені проєкту на основі `.env.example` та вкажіть `APP_VERSION`.

## Локальний запуск

Для локального запуску вам знадобиться Python 3.12+ та менеджер пакетів `uv`.

1. Клонуйте репозиторій та перейдіть у директорію проєкту:
   ```bash
   git clone https://github.com/BogdanovychA/slovohr-ai.git
   cd slovohr-ai
   ```

2. Створіть віртуальне середовище та встановіть залежності:
   ```bash
   uv sync
   ```

3. Налаштуйте конфігурацію застосунку у `src/assets/.env`.

4. Запустіть застосунок:
   ```bash
   uv run python src/main.py
   ```

## Запуск через Docker

Застосунок можна розгорнути за допомогою Docker Compose. Внутрішній веб-сервер працюватиме на порту 8009.

1. Налаштуйте конфігурацію застосунку (`src/assets/.env`) та Docker (`.env` у корені).

2. Запустіть контейнер:
   ```bash
   docker compose up -d
   ```

Застосунок буде доступний за адресою `http://localhost:8009`.
